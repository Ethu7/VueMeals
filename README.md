# Vue Meal
Web app for meal prep that accesses API with recipes for keto, vegan and paleo diets
## Project setup
```
npm install vue
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
